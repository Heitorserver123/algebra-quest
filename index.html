O que é um Monômio?

Um monômio é uma expressão algébrica formada por um único termo, que pode ser um número, uma variável ou o produto de números e variáveis. A palavra "mono" vem do grego e significa um.

Forma geral: a · x^n
onde a = coeficiente (número real) e x^n = parte literal (variável com expoente)
Exemplos de Monômios

5x  |  -3y²  |  7ab  |  -2x³y  |  4  |  x  |  ½a²b³

Não são Monômios

x + y  (dois termos)
a - b²  (subtração de termos)
3x/y  (variável no denominador)

Partes do Monômio

Todo monômio é composto por duas partes principais:

ParteDescriçãoExemplo: 5x²y³
CoeficienteA parte numérica do monômio5
Parte LiteralAs variáveis com seus expoentesx²y³
GrauSoma dos expoentes da parte literal2 + 3 = 5
⚠️ Atenção!

O grau de um monômio numérico (apenas um número, como 7 ou -3) é zero. Já o monômio nulo (coeficiente = 0) não tem grau definido.

Grau do Monômio

O grau de um monômio é a soma dos expoentes de todas as variáveis que compõem a parte literal.

Exemplos de grau:
4x³ → grau 3 (só um expoente: 3)
-2x²y → grau 3 (2 + 1 = 3)
7a²b³c → grau 6 (2 + 3 + 1 = 6)
9 → grau 0 (monômio numérico)
x → grau 1 (expoente implícito = 1)

Monômios Semelhantes

Dois ou mais monômios são semelhantes quando possuem a mesma parte literal (mesmas variáveis com os mesmos expoentes). Apenas o coeficiente pode ser diferente.

✅ São semelhantes

3x² e -7x² → mesma parte literal x²
5ab e 2ab → mesma parte ab
4x²y³ e -x²y³ → mesma parte x²y³

❌ Não são semelhantes

3x² e 3x³ → expoentes diferentes
5ab e 5a²b → expoentes diferentes
2x e 2y → variáveis diferentes

Adição e Subtração de Monômios

Só é possível somar ou subtrair monômios semelhantes. Opera-se apenas os coeficientes, mantendo a parte literal.

Exemplos:
5x² + 3x² = (5+3)x² = 8x²
7ab - 2ab = (7-2)ab = 5ab
4x³ + 2x² → NÃO simplifica (termos diferentes)
-3x²y + 8x²y - x²y = (-3+8-1)x²y = 4x²y

Multiplicação de Monômios

Multiplica-se os coeficientes e somam-se os expoentes das variáveis iguais.

Regra: a^x · a^y = a^(x+y) (soma dos expoentes de mesma base)
Exemplos:
3x² · 4x³ = (3·4) · x^(2+3) = 12x⁵
2a · 5a²b = (2·5) · a^(1+2) · b = 10a³b
(-3x²y) · (2xy³) = -6 · x^(2+1) · y^(1+3) = -6x³y⁴
4ab² · (-ab) = -4 · a^(1+1) · b^(2+1) = -4a²b³

Divisão de Monômios

Divide-se os coeficientes e subtraem-se os expoentes das variáveis iguais.

Regra: a^x ÷ a^y = a^(x-y) (subtração dos expoentes de mesma base)
Exemplos:
12x⁵ ÷ 4x² = (12÷4) · x^(5-2) = 3x³
20a⁴b³ ÷ 5a²b = (20÷5) · a^(4-2) · b^(3-1) = 4a²b²
-15x³y² ÷ 3xy = -5 · x^(3-1) · y^(2-1) = -5x²y

Potência de Monômio

Eleva-se o coeficiente à potência e multiplica-se os expoentes das variáveis.

Regra: (a · x^n)^m = a^m · x^(n·m)
Exemplos:
(2x³)² = 2² · x^(3·2) = 4x⁶
(3ab²)³ = 3³ · a³ · b^(2·3) = 27a³b⁶
(-2x²y)³ = (-2)³ · x⁶ · y³ = -8x⁶y³
💡 Dica importante:

Quando o coeficiente é negativo e o expoente é par, o resultado é positivo. Quando o expoente é ímpar, o resultado é negativo. Ex: (-2)² = 4 e (-2)³ = -8.

O que é um Polinômio?

Um polinômio é uma expressão algébrica formada pela soma (ou diferença) de dois ou mais monômios não semelhantes. Cada monômio que compõe o polinômio é chamado de termo.

Forma geral: a_n·x^n + a_(n-1)·x^(n-1) + ... + a_1·x + a_0
onde a_n ≠ 0 e os coeficientes são números reais
Exemplos de Polinômios

x + 3 (binômio)
x² - 4x + 4 (trinômio)
2x³ + x² - 5x + 1
a² + 2ab + b²

Classificação por Nº de Termos

Monômio: 1 termo
Binômio: 2 termos
Trinômio: 3 termos
Polinômio: 4+ termos

Classificação por Grau

Grau 1: linear
Grau 2: quadrático
Grau 3: cúbico
Grau 4+: quártico, etc.

Grau do Polinômio

O grau de um polinômio é o maior grau dentre todos os seus termos. Analisa-se o grau de cada monômio e escolhe-se o maior.

Exemplos de grau:
3x² + 5x - 7 → grau 2 (maior expoente é 2)
x³ - 2x² + x - 1 → grau 3 (maior expoente é 3)
4x⁵ + x² - 8 → grau 5 (maior expoente é 5)
2ab² + 3a²b - a → grau 3 (2ab² tem grau 3: 1+2)

Reduzindo um Polinômio

Um polinômio está na forma reduzida quando não possui termos semelhantes. Para reduzi-lo, identifique e some os termos semelhantes.

Exemplo: reduzir 3x² + 5x - 2x² + 4 - x
Passo 1: identificar semelhantes
Termos em x²: 3x² e -2x²
Termos em x: 5x e -x
Termos livres: 4
Passo 2: somar cada grupo
= (3-2)x² + (5-1)x + 4 = x² + 4x + 4 ✅

Valor Numérico de um Polinômio

O valor numérico é o resultado obtido ao substituir a variável por um número específico.

Exemplo: calcule p(2) para p(x) = 3x² - 2x + 1
p(2) = 3·(2)² - 2·(2) + 1
p(2) = 3·4 - 4 + 1
p(2) = 12 - 4 + 1
p(2) = 9 ✅
Exemplo: calcule q(-1) para q(x) = x³ - 2x + 5
q(-1) = (-1)³ - 2·(-1) + 5
q(-1) = -1 + 2 + 5
q(-1) = 6 ✅

Adição de Polinômios

Agrupe e some os termos semelhantes.

Exemplo: (3x² + 5x - 2) + (x² - 3x + 7)
= 3x² + 5x - 2 + x² - 3x + 7
= (3x² + x²) + (5x - 3x) + (-2 + 7)
= 4x² + 2x + 5 ✅

Subtração de Polinômios

Inverta o sinal de todos os termos do subtraendo e depois some os semelhantes.

Exemplo: (5x² + 3x - 1) - (2x² - x + 4)
Passo 1: distribuir o sinal negativo
= 5x² + 3x - 1 - 2x² + x - 4
Passo 2: agrupar semelhantes
= (5x² - 2x²) + (3x + x) + (-1 - 4)
= 3x² + 4x - 5 ✅
⚠️ Cuidado com os sinais!

Ao subtrair, o sinal negativo muda o sinal de todos os termos do segundo polinômio, não apenas do primeiro. Ex: -(2x² - x + 4) = -2x² + x - 4.

Multiplicação de Polinômios

Use a propriedade distributiva: multiplica-se cada termo do primeiro polinômio por todos os termos do segundo.

Monômio × Polinômio: 2x · (3x² - 4x + 5)
= 2x·3x² + 2x·(-4x) + 2x·5
= 6x³ - 8x² + 10x ✅
Binômio × Binômio: (x + 3)(x + 2)
= x·x + x·2 + 3·x + 3·2
= x² + 2x + 3x + 6
= x² + 5x + 6 ✅
Trinômio × Binômio: (x² + 2x - 1)(x + 3)
= x²·x + x²·3 + 2x·x + 2x·3 + (-1)·x + (-1)·3
= x³ + 3x² + 2x² + 6x - x - 3
= x³ + 5x² + 5x - 3 ✅

Divisão de Polinômios

Segue o algoritmo da divisão longa — semelhante à divisão de números inteiros.

💡 Antes de dividir:

Ordene os polinômios de forma decrescente em relação ao grau e preencha graus ausentes com coeficiente 0.

Exemplo: (x² + 5x + 6) ÷ (x + 2)
Dividendo: x² + 5x + 6 | Divisor: x + 2
x² ÷ x = x → 1º termo do quociente
x · (x + 2) = x² + 2x
Resto parcial: (x² + 5x + 6) - (x² + 2x) = 3x + 6
3x ÷ x = 3 → 2º termo do quociente
3 · (x + 2) = 3x + 6
Resto final: (3x + 6) - (3x + 6) = 0
✅ Quociente = (x + 3), Resto = 0
Exemplo com resto: (2x² + 3x - 1) ÷ (x + 2)
2x² ÷ x = 2x → 1º termo
2x · (x + 2) = 2x² + 4x
Resto parcial: (2x² + 3x - 1) - (2x² + 4x) = -x - 1
-x ÷ x = -1 → 2º termo
-1 · (x + 2) = -x - 2
Resto final: (-x - 1) - (-x - 2) = 1
✅ Quociente = (2x - 1), Resto = 1

Identidade dos Polinômios

Dois polinômios são idênticos quando valem a igualdade para todos os valores da variável. Nesse caso, seus coeficientes correspondentes são necessariamente iguais.

Exemplo: encontre a, b e c
Se ax² + bx + c = 3x² - 5x + 2 para todo x
Então: a = 3, b = -5, c = 2 ✅

Raízes de um Polinômio

A raiz de um polinômio p(x) é o valor de x que faz p(x) = 0. Encontrar as raízes é fundamental em equações polinomiais.

Exemplo: verifique se x = 2 é raiz de p(x) = x² - 5x + 6
p(2) = (2)² - 5·(2) + 6
p(2) = 4 - 10 + 6
p(2) = 0 ✅ Logo, x = 2 É raiz!

O que são Produtos Notáveis?

Produtos notáveis são multiplicações especiais entre polinômios que surgem com tanta frequência na álgebra que recebem fórmulas próprias. Conhecê-las economiza tempo e é essencial para fatoração, equações e simplificações.

💡 Por que aprender?

Em vez de fazer a distributiva completa toda vez, você aplica a fórmula diretamente. São a base para fatoração polinomial e resolução de equações de grau superior.

1️⃣ Quadrado da Soma

Quadrado da Soma de Dois Termos
(a + b)² = a² + 2ab + b²

O quadrado da soma é igual ao quadrado do primeiro, mais o dobro do produto dos dois termos, mais o quadrado do segundo. Atenção: o sinal de 2ab acompanha o sinal de b — aqui ambos positivos.

Exemplos — Quadrado da Soma:
(x + 3)² = x² + 2·x·3 + 3² = x² + 6x + 9
(2a + 5)² = (2a)² + 2·2a·5 + 5² = 4a² + 20a + 25
(3x + y)² = 9x² + 6xy + y²
(x + 1)² = x² + 2x + 1
(4a + 3b)² = 16a² + 24ab + 9b²
⚠️ Erro clássico!

(a + b)² NÃO é a² + b². O termo do meio 2ab nunca pode ser omitido!

2️⃣ Quadrado da Diferença

Quadrado da Diferença de Dois Termos
(a - b)² = a² - 2ab + b²

Similar ao quadrado da soma, mas com o termo central negativo. O último termo, b², é sempre positivo, pois é o quadrado de b.

Exemplos — Quadrado da Diferença:
(x - 4)² = x² - 2·x·4 + 4² = x² - 8x + 16
(3a - 2b)² = 9a² - 12ab + 4b²
(5x - 1)² = 25x² - 10x + 1
(2x - y)² = 4x² - 4xy + y²
(a - 7)² = a² - 14a + 49

3️⃣ Produto da Soma pela Diferença

Produto da Soma pela Diferença (Diferença de Quadrados)
(a + b)(a - b) = a² - b²

O produto de uma soma pela diferença de dois mesmos termos resulta na diferença dos quadrados. O termo do meio sempre se cancela — é o mais elegante dos produtos notáveis e muito usado em simplificações e fatorações.

Exemplos — Produto da Soma pela Diferença:
(x + 5)(x - 5) = x² - 25
(3a + 2b)(3a - 2b) = 9a² - 4b²
(4x + 1)(4x - 1) = 16x² - 1
(2x + 7y)(2x - 7y) = 4x² - 49y²
(x² + 3)(x² - 3) = x⁴ - 9
💡 Como reconhecer?

Se você vir um binômio do tipo a² - b² (dois quadrados perfeitos com subtração), saiba que ele se fatora como (a+b)(a-b).

4️⃣ Cubo da Soma

Cubo da Soma de Dois Termos
(a + b)³ = a³ + 3a²b + 3ab² + b³

O cubo da soma possui quatro termos, todos com sinal positivo. Os coeficientes seguem o padrão 1, 3, 3, 1 (triângulo de Pascal). Os expoentes de a decrescem (3,2,1,0) e os de b crescem (0,1,2,3).

Exemplos — Cubo da Soma:
(x + 2)³ = x³ + 3x²·2 + 3x·4 + 8 = x³ + 6x² + 12x + 8
(a + 1)³ = a³ + 3a² + 3a + 1
(2x + y)³ = 8x³ + 12x²y + 6xy² + y³
(x + 3)³ = x³ + 9x² + 27x + 27

5️⃣ Cubo da Diferença

Cubo da Diferença de Dois Termos
(a - b)³ = a³ - 3a²b + 3ab² - b³

Idêntico ao cubo da soma, mas os sinais se alternam: +, -, +, -. Atenção especial: o último termo é -b³ (negativo!).

Exemplos — Cubo da Diferença:
(x - 3)³ = x³ - 9x² + 27x - 27
(a - 1)³ = a³ - 3a² + 3a - 1
(2x - y)³ = 8x³ - 12x²y + 6xy² - y³
(x - 2)³ = x³ - 6x² + 12x - 8

6️⃣ Soma e Diferença de Cubos (Fatoração)

Fatoração: Soma e Diferença de Cubos
a³ + b³ = (a + b)(a² - ab + b²)
a³ - b³ = (a - b)(a² + ab + b²)

Usados para fatorar expressões cúbicas. O trinômio do segundo fator nunca é um quadrado perfeito (não confunda!). O sinal do termo ab no trinômio é sempre oposto ao sinal entre os cubos.

Exemplos — Soma e Diferença de Cubos:
x³ + 8 = x³ + 2³ = (x + 2)(x² - 2x + 4)
27a³ - 1 = (3a)³ - 1³ = (3a - 1)(9a² + 3a + 1)
8x³ + y³ = (2x + y)(4x² - 2xy + y²)
125 - x³ = 5³ - x³ = (5 - x)(25 + 5x + x²)

Como Reconhecer Produtos Notáveis

Muitas vezes você recebe um polinômio já expandido e precisa reconhecer que ele é um produto notável para fatorar. Veja os critérios:

É quadrado perfeito?

Trinômio com 3 termos: verifique se o 1º e 3º são quadrados perfeitos e se o 2º é exatamente o dobro do produto das raízes. Ex: x² + 6x + 9 = (x+3)²

É diferença de quadrados?

Binômio: dois quadrados perfeitos separados por subtração. Ex: x² - 16 = (x+4)(x-4)

É cubo perfeito?

Quadrinômio: verifique se o 1º e 4º termos são cubos perfeitos e se os coeficientes seguem o padrão 1,3,3,1. Ex: x³ - 6x² + 12x - 8 = (x-2)³

É soma/diferença de cubos?

Binômio: dois cubos perfeitos com soma ou subtração. Ex: 8x³ - 1 = (2x)³ - 1³ = (2x-1)(4x²+2x+1)

Exercício completo: fatore x⁴ - 16
x⁴ - 16 = (x²)² - (4)² → diferença de quadrados
= (x² + 4)(x² - 4) → aplicar novamente no 2º fator
= (x² + 4)(x + 2)(x - 2) ✅ completamente fatorado
Exercício completo: fatore 2x³ - 16
2x³ - 16 = 2(x³ - 8) → fator comum primeiro
= 2(x³ - 2³) → diferença de cubos
= 2(x - 2)(x² + 2x + 4) ✅
🏆 Dica final:

Sempre verifique se o polinômio pode ser fatorado mais de uma vez! Um produto notável pode conter outro produto notável dentro de si. E nunca esqueça de tirar o fator comum antes de aplicar qualquer fórmula.
