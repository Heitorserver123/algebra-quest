O que é um Monômio?

Um monômio é uma expressão algébrica formada por um único termo, que pode ser um número, uma variável ou o produto de números e variáveis.

Forma geral: a · x^n
onde a = coeficiente e x^n = parte literal

Partes do Monômio

Todo monômio tem duas partes: coeficiente (número) e parte literal (variáveis).

Exemplo: 5x²y³
Coeficiente: 5
Parte literal: x²y³
Grau: 2 + 3 = 5

Operações com Monômios

Adição e Subtração: Só é possível com monômios semelhantes (mesma parte literal).

3x + 2x = 5x ✅
3x + 2y = não é possível ❌

Multiplicação: Multiplica coeficientes e soma expoentes.

2x² · 3x = 6x³

O que é um Polinômio?

Um polinômio é a soma de dois ou mais monômios.

Exemplos:
x + 2 (binômio)
x² + 2x + 1 (trinômio)

Grau de um Polinômio

É o maior expoente entre todos os termos.

x² + 3x + 2 → grau 2
x³ + x² + 1 → grau 3

Operações com Polinômios

Adição: Soma os termos semelhantes.

(x² + 2x) + (3x² + x) = 4x² + 3x

Multiplicação: Usa a propriedade distributiva (chuveirinho).

(x + 2)(x + 3) = x² + 3x + 2x + 6 = x² + 5x + 6

Produtos Notáveis

São multiplicações especiais que seguem padrões. Conhecê-los acelera os cálculos!

1. Quadrado da Soma

(a + b)² = a² + 2ab + b²
(x + 3)² = x² + 2(x)(3) + 3² = x² + 6x + 9

2. Quadrado da Diferença

(a - b)² = a² - 2ab + b²
(x - 2)² = x² - 2(x)(2) + 2² = x² - 4x + 4

3. Produto da Soma pela Diferença

(a + b)(a - b) = a² - b²
(x + 5)(x - 5) = x² - 25
